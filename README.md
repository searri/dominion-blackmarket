# Black Market Card Randomizer

<a href="https://wiki.dominionstrategy.com/index.php/Black_Market" target="_blank"><i>Read more on the Dominion Wiki</i></a>

![](./Black_Market.jpg)

**NOTE**: This randomizer is only loaded with the cards _I_ have in my supply. To add/remove cards, modify the JSON files in [`cards`](./cards/) and re-run [`merge_files.py`](./merge_files.py).

Serve locally with:
```
python -m http.server
```

I grabbed the card YAML files from [here](https://github.com/blakevanlan/KingdomCreator/tree/master/sets) and the card images from [here](https://github.com/blakevanlan/KingdomCreator/tree/master/docs/img/cards).
